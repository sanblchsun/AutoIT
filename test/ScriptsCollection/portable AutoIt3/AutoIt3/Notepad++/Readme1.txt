stylers.xml и stylers.model.xml - стандартный стиль Notepad++, но с изменёным цветовым стилем для au3
themes\AutoIt_Def.xml - дополнительный цветовой стиль совпадающий со стилем оригинального SciTE с голубым фоном и жёлтым выделением строк.
themes\AutoIt+Zenburn.xml - дополнительный цветовой стиль на тёмном фоне цветастый шрифт, более мягкий для глаз.

стандартные темы можно переименовать и скопировать в themes, чтоб не заменять оригинальную дефолтную Notepad++
Стиль выбирать в Опции > Определение стилей > Раскрывающийся список "Выбрать стиль", выбираем имя файла стиля.


langs.xml - содержит в тегах <Language name="autoit" ext="au3" ...>    </Language>  типы и ключевые слова, для которых применяется цвет.

autoit.xml, batch.xml - содержат список команд, которые всплавают в виде справочного списка, в котором выбрать стрелками+Enter. Для включения автоподстановки в Notepad++ выбираем Опции > Настройки > Резерв/Автодополнение > Поставить галку "Включить для каждого ввода", Отметить "Завершение функций", Поставить галку "Подсказка при вводе параметров функций".

Скачайте плаг XBrackets.dll для автозакрытия скобок {([ ])}
Скачайте плаг NppExec.dll для выполнения команд по F6, в комплекте сохранённые команды в файле \plugins\Config\npes_saved.txt, а в папке AutoIt3 три скрипта (с описанием) для выполнения из Notepad++.
По F5 можно задать те же комманды что и по F6, но без вызова промежуточного диалога. Сохраняются команды в shortcuts.xml


Переменные Notepad++, которые удобно использовать по горячим клавишам добавляемым по F5
$(FULL_CURRENT_PATH)  : E:\my Web\main\welcome.html  - полный путь к файлу, который открыт в Notepad++
$(CURRENT_DIRECTORY)  : E:\my Web\main\  - текущий каталог файла, который открыт в Notepad++
$(FILE_NAME)  : welcome.html  - имя файла, который открыт в Notepad++
$(NAME_PART)  : welcome  - имя файла без расширения, который открыт в Notepad++
$(EXT_PART)  : html - расширение файла, который открыт в Notepad++

$(NPP_DIRECTORY)  : путь к notepad++.exe
$(CURRENT_WORD)  : выделенное слово Notepad++.
$(CURRENT_LINE)  : строка, где установлен текстовый курсор в Notepad++.
$(CURRENT_COLUMN)  : колонка, где установлен текстовый курсор в Notepad++.

Горячие клавиши:
Ctrl+Q - закомментировать / раскомментировать выделенный блок
Alt+Shift+Стрелка вверх / вниз - выделив текст и примених горячую клавишу выбеляем текст колонкой. Позволяет удалить к примеру цифры в начале каждой строки.
Tab, Shift+Tab - двигать выделенный блок влево / вправо с добавлением табуляции.
Ctrl+D - дублировать строку, иногда полезно продублировать сроку с командой и слегка подправить её.
Ctrl+Shift+Стрелка вверх / вниз - двигать строку
Ctrl+W - Закрыть документ. Отключил эту клавишу по причине частого промаха вместо Ctrl+Q
Двойной клик по слову - выделяет его. Следующий двойной клик выделяет строку.
Выделенный текст можно перетягивать удерживая левую кнопку мыши, тескт вставляется при отпускании в позиции курсора.

Ниже список комбинаций используемый во всех программах
Ctrl+Shift+Стрелка влево / вправо - выделять текст по словам
Shift+Стрелка влево / вправо - выделять текст по буквам
Shift+Home / End - выделять текст до начала / конца строки
Shift+PageUp / PageDown - выделять текст до начала / конца всего текста

Ctrl+A - выделить всё, применяется перед комбинацией Ctrl+C для копирования текста
Ctrl+S - сохранить
Ctrl+Z / X / C / V - отменить / вырезать / копировать / вставить
