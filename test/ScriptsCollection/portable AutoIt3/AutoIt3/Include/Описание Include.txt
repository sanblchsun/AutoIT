Некоторые скрипты требуют функции находящиеся в дополнительных библиотеках функций. Чтобы скрипт не вылетал с ошибкой, смотрим в заголовке скрипта необходимые ему библиотеки, например

#Include <WinAPIEx.au3>

то есть требуется наличие файла WinAPIEx.au3 в папке Include либо рядом со скриптом. При компиляции библиотека добавляется в скомпилированный скрипт. Существуют набор стандартных библиотек, и библиотек созданных пользователями AutoIt3. Отличие библиотеки от скрипта заключается в том, что в библиотеке находятся только функциии, описание их действий.

Вот дополнительные UDF (User Defined Function Notes - пользовательские функции)

_XMLDomWrapper.au3 - работа с форматом XML
audio.au3 - управление для медиаплеера, регулировка регуляторов микшера
BigNum.au3 - для выполнения рассчётов с большими числами (использовать в калькуляторах)
ColorPicker.au3
ExpListView.au3 - экслорер в виде ListView (3.3.5.0)
GDIP.au3 - графика
GDIPConstants.au3 - константы для GDIP.au3
GraphGDIPlus.au3 - построение графиков
GDIpProgress.au3 - прогресс бар основанный на картинках
GUICtrlSetOnHover.au3
GUICtrlSetOnHover_UDF.au3
GUIScroll.au3 - Gui с прокруткой (scroll)
hash.au3 - вычисление констрольных сумм MD5 и т.д.
HKCUReg.au3 - операции с реестром. Необходимы Reg.au3 и Privilege.au3
HotKey.au3 - горячие клавиши
HotKey_17b.au3
HotKeyInput.au3 - диалог назначения горячих клавиш
HotKeySelect.au3
Icons.au3
ID3.au3 - UDF для чтения тегов в *.mp3 файлах.
IsPressed_UDF.au3 - проверка нажатий клавиш клавиатуры
ListView_Progress.au3 - отобразить прогресс в  ListView
Midiudf.au3 - для воспроизведения midi-файлов
ModernMenuRaw.au3
Privilege.au3 - необходим для Services.au3
Reg.au3 - загрузить/выгрузить кусты реестра. Необходим Privilege.au3
Registry_UDFs.au3 - работа с реестром
resources.au3 - добавление ресурсов в скомпилированный скрипт и их использование
Services.au3 - управление службами
SoundGetSetQuery.au3 - регулировка регуляторов микшера
SysTray_UDF.au3 - позволяет осуществить клик на иконке в трее
TIG.au3 - создание анимационной иконки в трее
ToolTip_UDF.au3 - разноцветные подсказки с форматированным текстом и иконками
vkArray.au3 - для HotKeyInput.au3
vkConstants.au3 - константы
WinAPIEx.au3 - использование ресурсов WindowsXP
WMMedia.au3 - управление для медиаплеера
