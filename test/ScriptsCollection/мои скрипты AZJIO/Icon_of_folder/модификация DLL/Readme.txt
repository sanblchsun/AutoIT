

Для быстрой модификации Folder_ico.dll, если хочется создать свою тему, то нужно создать в текущей папке 24 иконки с именами 1, 2...24 и добавить программку ResHacker.exe. Далее выполняем Folder_ico_modify__dll.bat и Folder_ico.dll уже с новыми иконками. Добавляем Folder_ico.dll в каталог утилиты Icon_of_folder с новым именем, если сделать подмену default.dll, то тема будет стартовой по умолчанию. Folder_ico.dll можно использовать переименовав одну из готовых тем.
