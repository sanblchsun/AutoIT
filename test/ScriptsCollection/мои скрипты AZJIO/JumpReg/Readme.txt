
Тех-информация по исходнику JumpReg.au3

Компилировать используя AutoIt3Wrapper. При компиляции выдаст 2 предупреждения на функции Assign, IsDeclared, не обращать внимания, так как переменные добавлены в игнорирование обфускации.

JumpReg_CmdLine - урезанный вариант, только для прыжка с ком-строки, без истории и каких либо настроек.
