

Search_duplicates.au3 - основной исходник

Search_duplicates (Group).au3 - дополнительный вариант с использованием UDF. Группы выглядят красивее, без чекбокса, но проблема с изменением цвета и скорость построения списка при 3000 файлов 4 секунды против 2-х (и это без установко цвета).

Search_duplicates.dll - содержит иконки; при использовании нескомпилированного скрипта иконки берутся из неё.
