
The_generator_of_melodies - музыкальная программка, которая поможет создать мелодии для динамика ПК.
Скрипты можно выполнять с помощью командной строки (пример Start_melodies.bat) или встроить в свои скрипты программы построенные на AutoIt3.

Обновления
v0.4
Добавлена возможность использовать в скомпилированном виде без установленного AutoIt3
Добавлена кнопка "Открыть", "Стоп"
Если среди языковых раскладок присутствует русский язык, то автоматически назначается русский интерфейс, иначе английский.
Добавлены горячие клавиши
